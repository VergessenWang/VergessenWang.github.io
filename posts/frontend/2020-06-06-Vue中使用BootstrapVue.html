项目运行报错: System limit for number of file watchers reached

原来是linux系统中限制了用户可以watch文件变更的最大数，因此需要通过sysctl.conf进行更改数值

echo fs.inotify.max_user_watches=524288 | sudo tee -a /etc/sysctl.conf && sudo sysctl -p
